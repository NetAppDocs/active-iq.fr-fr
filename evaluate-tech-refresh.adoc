= Déterminez si une actualisation du système est recommandée pour votre système de stockage dans Digital Advisor
:hardbreaks:
:toclevels: 1
:allow-uri-read: 
:toclevels: 1
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Si vous souhaitez déterminer si une actualisation du système est recommandée pour votre contrat de Tech Support ou votre matériel, vous pouvez utiliser l'option d'actualisation du système.

Vous pouvez accéder à cette fonctionnalité soit via la planification du cycle de vie (*Storage* > *Lifecycle planning*), soit via Digital Advisor (*Dashboard > Planning widget > System Refresh*).

Pour plus d'informations sur cette fonctionnalité, consultez link:https://docs.netapp.com/us-en/bluexp-economic-efficiency/use/tech-refresh.html["Évaluer une actualisation du système"] dans la documentation de la console.
